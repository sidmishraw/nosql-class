To create library database in SQLite

  1. Have sqlite3.exe ready in your ssytem. sqlite3 is the command line shell to use sqlite.
  2. Downlaod SQL DDL program and book.csv, loan.csv, user.csv The program will be used to that will create and populate user, book and loan tables in the library database.
  3. Run sqlite3 as shown below to create library database. 
    %sqlite3 library.db
sqlite> .databases
  4. Execute the library_sqlite.sql script on sqlite prompt and check if tables are created. 
    sqlite>.read library_sqlite.sql
sqlite>.tables
  5. Now you can manipulate these tables using various SQL commands. Also, you can create views and insteadof triggers.
